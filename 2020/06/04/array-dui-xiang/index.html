Array对象

Array构造函数

通过Array对象创建数组有两种方式


let array = new Array(3);
let array2 = Array(3);

两种方式效果是一样的

  1. 如果参数n是一个正整数,那么创建一个数组长度为n的数组,数组的每一个位置是空值,没有元素的属性值(key);数组的成员都是空位,读取元数返回undefined,虽然可以读取length属性,但是取不到键名
  2. 如果参数n是一个负数或者是一个非正整数,那么将会报错;
  3. 如果参数是一个非数字值,那么创建的数组元素的第一个元素就是这个参数,这里参数便不会有什么转变;
  4. 如果是多个参数,那么将会创建一个数组,并且这个数组的元素是传入的参数;length属性等于传入参数的个数;

Array.isArray()方法

Array对象的我isArray方法可以判断参数是不是一个数组,如果是返回true,不是返回false;

Array实例的方法

valueOf()方法

valueOf()方法是每一个引用类型都有的方法,因为每一个对象都是Object的实例;Object.prototype中有valuOf()方法;通常valueOf()方法返回对象本身;

toString()方法

toString()方法将数组以string类型输出,它会遍历数组的每一个元素,如果元素是空值,null,undefined,那么会输出空字符串;如果元素是一个对象,会调用对象的toString()方法;如果元素还是一个
数组那么会调用数组的toString()方法;

push()方法

push()方法将参数添加到数组的后端,并返回数组的length长度,它会改变元素组;方法接收多个参数;每一个参数添加到数组后端;

pop()方法

pop()方法删除最后一个元素,并且返回删除的元素,如果数组是空数组,返回undefined,会改变原数组

shift()

shift()方法删除第一个元素,并且返回删除的元素,如果数组是空数组,返回undefined,会改变元素组

unShift()

unShit()方法在数组前端添加元素,返回数组的长度,参数可以是多个,会改变原数组

join()

join() 以参数为分隔符将数组元素分开;

concat()

concat()将参数与数组连接起来,如果参数是一个数组,那么将会将这个数组中的元素添加到新数组中,如果参数不是数组,会将参数添加到新数组中;它不会改变数组,返回新数组,它是数组的浅拷贝

reverse()

reverse()颠倒原数组,会改变数组

slice()

slice() 不会改变原数组;
参数一大于参数二,参数一大于等于数组length,返回空数组;参数是负数,从后面开始,返回新数组

splice,

第一个参数: 开始的位置
第二个参数: 删除的元素个数;
后面参数,添加的元素

sort()

sort()排序数组,返回大于0的数,交换元素;改变元素组

map()

map()第二个参数,改变回调函数的this指向; 忽略空元素

forEach()

forEach()第二个参数,改变回调函数的this指向; 忽略空元素

filter()

filter()第二个参数,改变回调函数的this指向; 忽略空元素

some()

some()第二个参数,改变回调函数的this指向; 忽略空元素

every()

every()第二个参数,改变回调函数的this指向; 忽略空元素;
