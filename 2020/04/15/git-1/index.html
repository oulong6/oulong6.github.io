git基础

git
git
发布日期:   2020-04-15

初始化本地仓库

1.通过这个指令让git管理这个本地仓库

git init

2.或者通过克隆已存在的项目仓库来初始化本地仓库

git clone url

添加文件到仓库

git add filename|.

撤销添加到暂存区的文件

git reset HEAD

撤销变动的内容,还未添加如暂存区中

git checkout --

提交暂存区中的内容

git commit -m '提交的信息'

查看Git日志

git log

查看提交的commit信息一行显示

git log --pretty=oneline

用图形的方式查看

git log --graph

git 将暂存区中的文件撤销到工作区

git add x.txt
git reset HEAD x.txt

将已修改的文件恢复到最后commit时的状态

git checkout -- 
git checkout -- . 恢复工作区中的全部的修改到最后一次commit时的状态
