css浮动及清除浮动

浮动元素是通过设置float属性为left或者right变成浮动元素；浮动元素是脱离文档流的，所以元素在文档流中的限制被打破，比如内联元素可以设置宽高；浮动元素的父元素的高度将会不包含浮动子元素的高度。

浮动元素的浮动步骤

    <div class="box">
        <div class="item" style="background-color: pink"> item1</div>
        <div class="item" style="background-color: #4b9149">item2</div>
        <div class="item" style="background-color: #d6d44f">item3</div>
        <div class="item" style="background-color: #cb0746">item4</div>
    </div>
    /* css */
     .box {
            width: 400px;
            height: 500px;
            background-color: gray;
        }
        .item {
            width: 50px;
            height: 50px;
        }


在没有给item元素设置浮动时，因为它们是块级元素是独占一行，一个接一个的排列

设置第一个子元素向left浮动

    .item:nth-of-type(1){
        float: left;
    }
    .item:nth-of-type(2){
        width: 80px;
    }


浮动过程抽象的描述：

item1在布局时，因为item1设置了float：left；在文档流中并不存在，它属于浮动流；所以文档流的布局方式不适合它，后面的元素，是属于文档流的，它们会一个及一个的排列；item1元素不在文档流中，所以后面的元素会占据原来item1元素的位置。图中的item的右边是不是有一个绿色的元素，因为item1是浮动元素了，不再文档流里，所以后面的元素会向上移动，占据原来item1的位置，item1元素下面的就是item2

      .item:nth-of-type(1){
            float: left;
        }
        .item:nth-of-type(2){
            float: left;
        }
        .item:nth-of-type(3){
            float: right;
        }
        .item:nth-of-type(4){
            float: right;
            clear:

        }

给元素设置左浮动或者右浮动时，元素会向左或者向右一个接一个的排列。类似于内联元素的排列方式；如果空间不够元素的大小，元素与换行直到空间足够。

清除浮动

清除浮动的属性是clear
1.clear: left; //左边不能有浮动元素，有换行
2.clear: right;//右边不能有浮动元素，有换行
3.clear: both;//左右两边边不能有浮动元素，有换行
如：在给元素添加clear: left;属性是，是要求元素的左边是不能有浮动的，如果该元素的左边有浮动元素那么该元素将会换行。cleat并不是清除浮动元素，而是要求设置这个属性的元素的右边或者右边不可以出现浮动元素，如果有浮动元素，那么当前元素会换行显示。浮动元素是按照顺序一个一个的布局的，当前浮动元素的后面浮动元素并不会影响当前的浮动元素；

清除浮动的抽象动画

清除浮动就是让设置clear属性的元素，如果他的左边或者右边有浮动元素那么该元素就会向下移动，直到没有浮动元素为止，从而使父元素的高度撑开了。
